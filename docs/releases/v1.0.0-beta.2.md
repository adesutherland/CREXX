# CREXX 1.0.0 Beta 2 Release Notes

_Release tag: `v1.0.0-beta.2`_

CREXX 1.0.0 beta 2 is the next Release 1 beta baseline after
`v1.0.0-beta.1`. This page is the working release note for changes that land
on `develop` during the beta 2 cycle.

## Planned Focus

- Carry forward the beta 1 toolchain, runtime, documentation, and release
  packaging baseline.
- Ship signed Windows release assets alongside the existing signed and notarized
  macOS packages.
- Capture beta 2 fixes, compatibility notes, and known limitations as they are
  completed.
