# Beta 3 Design And Progress Notes

Use this directory for durable notes that support beta 3 GitHub issues.

Prefer one note per issue or tightly related issue group. Keep GitHub as the
task tracker; keep architecture decisions, progress summaries, source-mapping
examples, test evidence, and release guidance here when they should be versioned
with the source tree.

Suggested filename pattern:

```text
<issue-number-or-short-slug>.md
```
